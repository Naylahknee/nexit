# Nexit brand assets

These runtime assets are derived only from the owner-approved PNG artwork supplied for the app. No logo geometry was redrawn or AI-generated.

- `nexit-wordmark-light.png`: navy and gold wordmark for light surfaces.
- `nexit-wordmark-dark.png`: white and gold wordmark for dark surfaces.
- `nexit-wordmark-split.png`: retained alternate treatment; not used by default.
- `nexit-primary-dark.png`: detailed primary butterfly-globe on navy.
- `nexit-icon-dimensional-source.png`: approved dimensional icon source.
- `nexit-icon-flat.png`: retained flat icon alternative.
- `app-icon-*.png`: opaque, full-bleed installable icon exports.
- `favicon-*.png` and `favicon.ico`: transparent butterfly favicon exports.

The composite brand board and favicon source sheet are references only and are intentionally excluded from runtime assets.
